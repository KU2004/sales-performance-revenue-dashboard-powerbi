# 📊 Sales Performance & Revenue Growth Dashboard | Power BI

## 🚀 Project Overview

This project presents an interactive Power BI dashboard designed to analyze **sales performance**, **revenue growth**, and **business KPIs**.
It helps businesses make **data-driven decisions** by transforming raw sales data into meaningful insights.

---

## 🎯 Objective

* Track total revenue, profit, and units sold
* Analyze target vs actual sales performance
* Monitor growth trends over time
* Identify top-performing regions, products, and sales executives

---

## 🛠 Tools & Technologies Used

* Power BI
* DAX (Data Analysis Expressions)
* Power Query
* Microsoft Excel

---

## 📂 Dataset Description

The dataset contains realistic sales data including:

* Date, Month, Quarter, Year
* Region and City
* Sales Executive Name
* Product Category and Product Name
* Units Sold, Selling Price
* Revenue, Cost, Profit
* Profit Margin %
* Customer Type (New/Existing)
* Sales Channel (Online/Offline)
* Target Sales vs Actual Sales
* Achievement % and Growth %

---

## 🔧 Power Query Transformations

* Removed null values and cleaned data
* Changed data types (Date, Currency, Numeric)
* Created Month, Quarter, and Year columns
* Removed duplicates
* Renamed columns for clarity

---

## 🧠 DAX Measures Used

* Total Revenue
* Total Cost
* Total Profit
* Profit Margin %
* Total Units Sold
* Target Sales
* Actual Sales
* Achievement %
* Growth %

---

## 📊 Dashboard Features

### 🔝 KPI Cards

* Total Revenue
* Total Profit
* Total Units Sold
* Achievement %
* Growth %

### 📈 Visualizations

* Revenue by Region (Column Chart)
* Product Category Performance (Bar Chart)
* Monthly Revenue Trend (Line Chart)
* Sales Channel Distribution (Donut Chart)
* Target vs Actual Sales (Clustered Column Chart)

### 📋 Data Table

* Region-wise Revenue, Profit, Achievement %

### 🎛 Filters (Slicers)

* Region
* Product Category
* Sales Executive
* Month
* Sales Channel

---

## 📈 Key Insights

* Identify top-performing regions and categories
* Analyze whether sales targets are achieved
* Understand monthly growth trends
* Compare online vs offline sales performance
